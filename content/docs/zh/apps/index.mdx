## 集成指南

大多数应用只需要配置以下信息即可连接 New API：

1. **API 地址**：您的 New API 服务地址
2. **API Key**：在 New API 控制台生成的令牌
3. **模型名称**：选择需要使用的模型

具体配置步骤请参考各应用的详细文档。
