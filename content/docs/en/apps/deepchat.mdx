## NewAPI Integration Method

### Method 1: One-Click Import

1. In the New API console sidebar, open **Token Management** and find the token you want to use
   <img src="/assets/deepchat/1-Enter-API-key.png" alt="Open Token Management" width="240" />

2. Select **DeepChat** from the token actions
   <img src="/assets/deepchat/2-API-key-deepchat.png" alt="Select DeepChat" width="360" />

3. Let the browser launch DeepChat and confirm opening the app
   ![Open DeepChat](/assets/deepchat/3-open-deepchat.png)

4. Confirm the configuration in DeepChat's Provider import preview
   ![Confirm DeepChat Configuration](/assets/deepchat/4-deepchat-ok.png)

5. Open the model list and select a New API model to start using it
   ![Open Model List](/assets/deepchat/5-open-models.png)

### Method 2: Manual Import

1. Copy the API Key and site address from the New API console
   ![New API Manual Configuration Guide](/assets/deepchat/7-new-api-manual-guidance.png)

2. In DeepChat settings, add or edit the New API Provider, then fill in the API address and API Key
   ![DeepChat Manual Configuration Guide](/assets/deepchat/6-deepchat-manual-guidance.png)

3. Open the model list and select a New API model to start using it
   ![Open Model List](/assets/deepchat/5-open-models.png)
