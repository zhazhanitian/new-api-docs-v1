## Integration Guide

Most applications only require configuring the following information to connect to New API:

1.  **API Address**: Your New API service address
2.  **API Key**: The Token generated in the New API console
3.  **Model Name**: Select the model to use

For specific configuration steps, please refer to the detailed documentation of each application.
